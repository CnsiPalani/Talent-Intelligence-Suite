# Talent-Intelligence-Suite
AI-Powered Talent Intelligence &amp; Workforce Optimization Suite (T-IQ)

## Overview
This project is a comprehensive HR analytics and automation platform built with Python. It includes modules for:
- Employee attrition prediction
- Performance regression
- Sentiment analysis
- Resume and job description matching
- Interview summarization
- Fraud detection (CNN)
- Exploratory data analysis (EDA)
- Time series forecasting
- Chatbot and LLM integration

The main user interface is a Streamlit dashboard (`streamlit_app.py`) that brings together these features for interactive use.

## Getting Started

### Prerequisites
- Python 3.8+
- Recommended: Create a virtual environment

### Installation
1. Clone the repository or copy the project files.
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. (Optional) Set up environment variables in `hrm.env`.

### Running the App
To launch the Streamlit dashboard:
```sh
streamlit run streamlit_app.py
```
